Select an element and return the text value.

> **Hint** Xpath can be used to return attributes!

## Options

Doesn't provide options.

## Usage

**DOM**

```
field: title
type: text
selector: h1#page-title
```

**Xpath**

```
field: title
type: text
selector: //*/h1
```
